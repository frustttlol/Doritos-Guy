What we collect

This website currently does not collect personal information through its "Dashboard Login" page. Clicking the "Continue with Discord" button simply redirects you to Discord and does not authenticate you with this website.

If you choose to invite and use the Doritos Guy Discord bot, the bot may process information provided by Discord's API in order to function. This can include:

  • Your Discord User ID
  • Server (Guild) ID
  • Channel and Role IDs when required for bot features
  • Information needed to execute commands or moderation features

The bot does not collect your Discord password, payment information, or private messages unless you directly interact with the bot where required.
